OpenClaw Installation in Proxmox VM

Überblick

Diese Anleitung beschreibt die einfachste Methode, OpenClaw in einer Proxmox VM zu installieren. OpenClaw ist eine AI-Agent-Plattform, die es ermöglicht, automatisierte Workflows und Integrationen zu erstellen.

Voraussetzungen:
• Proxmox VE (Virtual Environment)
• Mindestens 4 GB RAM für die VM
• 20 GB freier Speicherplatz
• Internetverbindung

Schritt-für-Schritt Anleitung

1 VM in Proxmox erstellen

1.1 Neue VM anlegen

  1. Im Proxmox Dashboard auf Create CT oder Create VM klicken
  2. Für OpenClaw empfehlen wir eine Ubuntu 22.04 LTS VM
  3. Folgende Ressourcen zuweisen:
    • CPU: 2 Kerne
    • RAM: 4 GB
    • Storage: 20 GB

1.2 Netzwerk konfigurieren

Stellen Sie sicher, dass die VM eine IP-Adresse im Netzwerk erhält (DHCP oder statisch).

2 Ubuntu vorbereiten

Nach der VM-Erstellung, mit SSH verbinden:

ssh user@vm-ip-address

System aktualisieren:

sudo apt update && sudo apt upgrade -y

Erforderliche Pakete installieren:

sudo apt install -y curl git nodejs npm docker.io docker-compose
3 OpenClaw installieren

3.1 Installationsscript ausführen

Das einfachste ist die offizielle Installation:

curl -fsSL https://openclaw.dev/install.sh | bash
Hinweis: Falls das Script nicht verfügbar ist, manuelle Installation siehe Schritt 3.2

3.2 Manuelle Installation (Alternative)

# Node.js 22 installieren
curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash -
sudo apt install -y nodejs

# OpenClaw global installieren
sudo npm install -g openclaw

# OpenClaw initialisieren
openclaw init

# Konfiguration bearbeiten
nano ~/.openclaw/config.yaml
4 Konfiguration

4.1 Basis-Konfiguration

Die Konfigurationsdatei befindet sich unter ~/.openclaw/config.yaml:

# Beispiel-Konfiguration
gateway:
  port: 8080
  host: 0.0.0.0

channels:
  telegram:
    enabled: true
    token: "YOUR_BOT_TOKEN"
  
  discord:
    enabled: false
    token: "YOUR_DISCORD_TOKEN"

agents:
  enabled: true
  directory: ~/.openclaw/agents

4.2 Umgebungsvariablen (optional)

export OPENCLAW_API_KEY="your-api-key"
export OPENCLAW_WEBHOOK_SECRET="your-webhook-secret"
5 OpenClaw starten

5.1 Als Service einrichten (empfohlen)

# Systemd Service erstellen
sudo tee /etc/systemd/system/openclaw.service > /dev/null <

5.2 Status prüfen

sudo systemctl status openclaw

5.3 Logs anzeigen

sudo journalctl -u openclaw -f
6 Zugriff testen

OpenClaw ist nun unter folgender URL erreichbar:

http://VM-IP-ADDRESS:8080
Proxmox Port-Weiterleitung (optional):
Falls Sie von außerhalb des Proxmox-Netzwerks zugreifen möchten, konfigurieren Sie eine Port-Weiterleitung im Router oder nutzen Sie einen Reverse Proxy.

Nützliche Befehle

OpenClaw verwalten

# Service starten
sudo systemctl start openclaw

# Service stoppen
sudo systemctl stop openclaw

# Service neu starten
sudo systemctl restart openclaw

# Konfiguration neu laden
sudo systemctl reload openclaw

# Autostart deaktivieren
sudo systemctl disable openclaw

Updates

# OpenClaw aktualisieren
sudo npm update -g openclaw

# System aktualisieren
sudo apt update && sudo apt upgrade -y

Fehlerbehebung

Problem: Port bereits belegt

# Port 8080 prüfen
sudo lsof -i :8080

# Alternativen Port in config.yaml verwenden
gateway:
  port: 8081

Problem: Berechtigungsfehler

# Berechtigungen korrigieren
sudo chown -R $USER:$USER ~/.openclaw

Problem: Node.js Version zu alt

# Node.js Version prüfen
node --version

# Mindestens v22 erforderlich
# Aktualisierung siehe Schritt 3.2

Sicherheitshinweise

Wichtig:
• Ändern Sie Standard-Passwörter
• Verwenden Sie HTTPS für Produktivumgebungen
• Firewall-Regeln anpassen (nur benötigte Ports öffnen)
• Regelmäßige Backups der VM erstellen
